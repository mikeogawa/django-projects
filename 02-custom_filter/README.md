# Django Custom Filter

A Custom Django Parameter Filter.

## Feature
- Validates
- Specify Required Queries


Example
```
>>> python main.py
python main.py
{'a': 'string', 'b': 3}
{'a': 'string', 'b': 3, 'c': [1, 2, 3, 4, 5]}
{'a': 'string', 'b': 3, 'c': [1]}
c is a class IntListFilter got None
```
